## Thathumusa Funeral Stokvel

This project is a Next.js App Router application for member management, funeral claims, contribution tracking, meetings, loans, charity item distribution, and onboarding.

### Included in this starter

- Grey-and-white responsive marketing and dashboard UI inspired by the provided references
- Clerk-ready provider wrapper
- Convex schema scaffold for the proposed data model
- Paystack and SMS configuration placeholders
- Member dashboard, admin dashboard, charity flow, terms page, and onboarding guide

### Run locally

```bash
npm install
npm run dev
```

### Still needed before production

- `NEXT_PUBLIC_CONVEX_URL`
- Paystack keys
- SMS provider credentials
- Real admin and member seed data
- Domain when you are ready to publish
